ID Vs Class

What is a Class?

Class is a way to apply certain style or behavior to multiple elements so that they all look and behave the same way, helps with keeping the webpage conisitent. class are indentified with a period (.) followed by class name.

What is an ID?

An ID is a unique identifier that helps pinpoint and style a practicular element. While the page can have many elements, each element can only ahve one ID, and each page can only have one element with a particular ID. IDs are identified with a hash (#), followed by the ID name.

What are the best practices associated with using classes vs. iDs?

ID

  • Selects a single element

  • Is written with a hash symbol followed by the ID name

  • Can only be used to mark a single element in a page

Class

  • Selects a group of elements

  • Is written with a period followed by the class name

  • Can be used to mark multiple elements within a page
